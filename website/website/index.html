QwertyBot

The most minimalist bot on Discord.

Simple commands, easy setup and more!

We Provide The Best

Easy commands

The command is literally "#" but you can change it to whatever you want, either in the code or with the command!

Reliable servers

The servers are self hosted and there may be small downtimes but other than that, it is always online! You can host it yourself too if you are hardcore.

Easy setup

Clone the repo, then just run the python code after adding your bot token. It is that simple.
